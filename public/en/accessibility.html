Accessibility Statement

Last updated: 3.7.2026

FAMILYPASTPORT is committed to making its services accessible to the general public. We work to implement the Israeli Standard
(SI 5568) based on WCAG 2.1 guidelines at Level AA.

Accessibility level: AA (Israeli Standard SI 5568)

1. Accessibility Policy

We believe in equal rights and in every person's right to receive information and services independently and equally.
This website was designed and built with special emphasis on accessibility for people with disabilities.

2. Accessibility Features on the Website

  • Keyboard navigation support
  • Screen reader support
  • Appropriate color contrast
  • Flexible and scalable font sizes
  • Detailed image descriptions (Alt text)
  • Hierarchically structured headings
  • Descriptive and clear links
  • Zoom support up to 200% without loss of content

WCAG 2.1 Compliance

The website complies with Level AA of the WCAG 2.1 standard, which includes:

  • Perceivable: Information is presented in ways users can perceive
  • Operable: Interface components are operable
  • Understandable: Information and the user interface are understandable
  • Robust: Content can be interpreted by a variety of technologies

3. Supported Assistive Technologies

  • NVDA (Windows)
  • JAWS (Windows)
  • VoiceOver (Mac/iOS)
  • TalkBack (Android)
  • Dragon NaturallySpeaking
  • Switch navigation

4. Accessibility Testing

The website was tested using:

  • Automated accessibility testing tools
  • Manual testing with screen readers
  • Keyboard-only navigation
  • Color contrast testing
  • Testing with users with disabilities

5. Ongoing Improvements

We continuously work to improve the website's accessibility and update it in accordance with technological developments and user feedback.
